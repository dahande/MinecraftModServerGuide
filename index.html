マイクラmod鯖接続手順

対象: Java Edition
最終更新: 2025-09-26
CurseForge をダウンロード

Webサイト「CurseForge」をダウンロード。

Modpack（ZIP）をダウンロード

下のボタンをクリックして、必要なModpack一式を取得してください。ブラウザでブロックされた場合は許可してください。

ZIPをダウンロード
ファイル名: modpack.zip／サイズ: 約 200 MB
ハッシュ(SHA256): xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
Modpack を選択して起動

対象プロファイルを選び起動。初回は依存ModのDLがあります。

サーバーを追加（アドレス入力）

マルチプレイ → サーバーを追加 → アドレス入力 → 保存。

接続・リソースパック許可

要求が出たら「はい」を選択してください。

参加後の初期設定

キー設定・チャット設定などを調整します。

よくあるエラー：Modバージョン不一致／接続が切れる → クライアントとサーバーのMod一覧・ログを確認。
